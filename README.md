# Testers.ai CLI Releases

Official distribution channel for the Testers.ai CLI.

## Installation

### Option 1: Homebrew (Recommended for Developers)
```bash
brew install testers-ai/releases/testers
```

### Option 2: Manual Install (DMG)
1. Download and open the disk image (.dmg).
2. Double-click the Install Testers.ai CLI.pkg icon.
3. Choose your installation type:
  A. Install for all users (Requires Admin)
    Installs to: /Applications/testers
    Command location: /usr/local/bin/testers

  B. Install for me only (No Password)
    Installs to: ~/Applications/testers
    Command location: ~/bin/testers

Note: The installer automatically adds the testers command to your Terminal path.
